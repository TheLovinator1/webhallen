# Webhallen

Track changes on [webhallen.com](https://www.webhallen.com) with the help of GitHub Actions.

## Usage

- Run scrape.sh
  - `chmod +x scrape.sh && ./scrape.sh`

## Contributing

Feel free to add new things to track or improve the existing ones.

## Configuration

The following environment variables can be set:

- `PUSH_ENABLED` - Set to `true` if you want to push the changes.

Don't forget to set your git username and email:

- `git config --global user.name "LoviBot"`
- `git config --global user.email "99759169+LoviBot@users.noreply.github.com"`
